Когда камера сфокусировалась на объекте, пользователь получает возможность выполнять различные действия с объектом, например, просматривать справочную информацию.

Главная идея приложения - дать пользователю возможность загадать желание и бросить монетку на постамент скульптуры.

Виртуальную монетку можно приобрести через любую платежную систему
