Когда камера сфокусировалась на объекте, пользователь получает возможность выполнять различные действия с объектом, например, просматривать дополнительную информацию.

Главная идея приложения - загадать желание и бросить монетку на постамент скульптуры. Виртуальную монетку можно приобрести через любую платежную систему, доступную пользователю.
